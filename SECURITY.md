# Security Policy

## Supported Versions

We provide security updates for the following versions:

| Version | Supported |
| ------- | --------- |
| 3.x | :white_check_mark: |
| < 3.0 | :x: |

## Reporting a Vulnerability

**Please do not report security vulnerabilities through public GitHub Issues.**

If you discover a security vulnerability, please email us at **jeecgos@163.com** with the following details:

- A description of the vulnerability
- Steps to reproduce the issue
- Potential impact and affected versions
- Any suggested fix (optional)


## Security Updates

If we fix a vulnerability, we will announce it in the release notes. Keep your installation up to date to receive the latest fixes.
