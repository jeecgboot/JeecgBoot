# Security Policy / 安全策略

## Supported Versions / 支持的版本

Only the latest stable release builds are supported with security patches.

## Reporting a Vulnerability / 漏洞汇报

If you identify a security vulnerability in JeecgBoot (such as SQL injection in MyBatis query configurations, process engine flow control escapes, Spring security bypasses, or frontend Vue3 XSS flaws), please do **not** open a public issue.

Please report vulnerabilities privately by emailing the maintainers or using GitHub's private vulnerability reporting feature.

如果您在系统中发现安全漏洞（如 MyBatis SQL 注入、工作流执行引擎越权、Spring Security 鉴权绕过、或 Vue3 前端 XSS 等），请勿通过公开 Issue 报告，请使用 GitHub 的私有漏洞报告功能或直接发送电子邮件联系维护者。
